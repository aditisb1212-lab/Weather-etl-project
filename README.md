# 🌦️ Weather Data ETL Pipeline

## 📖 Description

This project extracts real-time weather data using an API, processes it using Python, and stores it in CSV format.

## ⚙️ Tech Stack

* Python
* Pandas
* REST API

## 🚀 Features

* Fetch real-time weather data
* Convert JSON to structured data
* Store data in CSV

## ▶️ How to Run

1. Install dependencies
2. Run `main.py`

## 📊 Output

* weather_data.csv file generated
